# Gerenciar Workspaces

Este documento explica como utilizar os Workspaces para isolar projetos e prevenir limites de armazenamento do navegador.

Após ler este guia, você será capaz de:
* Alternar, criar e excluir workspaces.
* Reordenar a lista de projetos.
* Entender o isolamento de dados.

---

## Entender o isolamento de dados

O navegador impõe um limite estrito de armazenamento por chave de dados (aproximadamente 5MB). Para evitar o erro de cota excedida (*Quota Exceeded*), a extensão fragmenta o banco de dados. 

Cada Workspace atua como um repositório isolado. Prompts, Anotações, Etiquetas e Credenciais de Nuvem criados em um Workspace não afetam nem consomem a memória dos demais.

---

## Gerenciar a lista de Workspaces

Você controla seus ambientes através do menu localizado no rodapé da Galeria. O botão sempre exibe o nome do Workspace ativo.

Para executar ações de gerenciamento:
1. Abra a Galeria.
2. Clique no nome do Workspace no rodapé para abrir o menu suspenso.
3. Execute uma das seguintes ações:
   * **Alternar:** Clique no nome de outro Workspace para carregar seus dados imediatamente.
   * **Criar:** Clique em **+ Novo Workspace** no final da lista e defina um nome.
   * **Renomear:** Clique no ícone de lápis ao lado do Workspace.
   * **Excluir:** Clique no ícone de lixeira. 

> ⚠️ **Atenção:** Excluir um workspace apaga permanentemente todos os dados associados a ele no seu computador local.

---

## Reordenar Workspaces

Para priorizar os projetos mais acessados, você pode alterar a ordem da lista.

Para reordenar:
1. Abra o menu suspenso de Workspaces.
2. Clique e segure o item desejado.
3. Arraste o item para cima ou para baixo.
4. Solte o botão do mouse. A extensão salva a nova ordem automaticamente.
